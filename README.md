# Operação Fluxo 8

Jogo web sobre os 8 desperdícios do Lean, com sessão individual por navegador e QR de acesso.

## Abrir no computador

Abra `index.html` no navegador.

## Usar QR com celulares na mesma rede

1. Entre nesta pasta.
2. Rode `node server.js`.
3. Abra o endereço mostrado no terminal.
4. Clique no botão de QR no topo do jogo.

Cada aparelho cria sua própria sessão automaticamente. O resultado pode ser exportado em CSV no fim do jogo.
